# Instamind

Instamind is an AI commerce assistant concept built on top of Swiggy Food and Instamart.

This repository currently contains a presentable MVP landing page that shows:
- the core product idea
- the user flow
- the system design
- the post-approval roadmap

## Local run

```bash
npm run dev
```

Then open:

```bash
http://localhost:3000
```

## Current MVP scope

The MVP demonstrates:
- AI-assisted grocery refill planning
- meal-aware ordering suggestions
- preference memory
- Swiggy MCP integration path
- explainable cart generation
